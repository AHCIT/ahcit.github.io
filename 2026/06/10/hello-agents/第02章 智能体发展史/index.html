第02章 智能体发展史

1. 演进总览

智能体发展遵循"问题驱动"的迭代逻辑：每一个新范式的出现，都是为了解决上一代范式的核心痛点，而新方案在带来能力飞跃的同时，也引入新的局限，为下一代范式埋下伏笔。

核心演进路径：符号主义 → 分布式智能 → 联结主义 → 强化学习 → 预训练 → LLM 智能体

2. 符号主义时代

2.1 物理符号系统假说（PSSH）

1976 年由纽厄尔和西蒙提出，是符号主义的理论基石：

  • 充分性论断：任何物理符号系统都具备产生通用智能的充分手段
  • 必要性论断：任何展现通用智能的系统，本质必然是物理符号系统

核心信念：智能的本质就是符号的计算与处理。

2.2 专家系统

核心目标：模拟人类专家在特定领域解决问题的能力。

组件 说明
知识库 存放领域知识，常用"IF-THEN"产生式规则表示
推理机 通用推理程序，支持正向链（数据驱动）和反向链（目标驱动）
用户界面 人机交互接口

MYCIN 系统（1970s，斯坦福）：

  • 辅助诊断细菌性血液感染，约 600 条规则
  • 创新：引入**置信因子（CF）**处理不确定性
  • 表现超过非专业医生，达到专家水平

2.3 SHRDLU（1968-1970）

在"积木世界"中通过自然语言交互的综合性智能体：

  • 自然语言理解：处理复杂指令、指代消解、上下文记忆
  • 规划与行动：自主规划动作序列（如先移开障碍物再放积木）
  • 记忆与问答：可回答世界状态、行为历史、行为动机

历史意义：首次将语言理解、推理规划、行动记忆集成于统一系统，奠定了"感知-思考-行动"闭环的基础。

2.4 符号主义的根本性挑战

挑战 说明
知识获取瓶颈 手工编码知识成本高、难规模化，隐性知识难以表达
常识问题 庞大常识背景无法穷举编码
框架问题 动态世界中高效判断不变状态的计算难题
系统脆弱性 规则之外的微小变化即导致系统失灵

3. 基于规则的聊天机器人：ELIZA

3.1 设计思想

1966 年魏泽鲍姆开发，模仿罗杰斯学派心理治疗师。核心技巧：从不正面回答，而是通过关键词匹配 + 规则转换，将陈述变为开放式提问。

3.2 算法流程

  1. 关键词识别与排序：按优先级匹配关键词
  2. 分解规则：用通配符捕获句子其余部分
  3. 重组规则：随机选择模板生成回应
  4. 代词转换：I→you, my→your 等

3.3 根本局限

  • 缺乏语义理解：不理解否定词等语义，机械匹配
  • 无上下文记忆：无状态，每次仅基于当前单句
  • 规则扩展性问题：规则爆炸式增长，冲突管理复杂

ELIZA 效应：用户因情感投射而相信机器理解自己，揭示了符号主义的核心矛盾——看似智能的表现完全依赖预设规则，没有真正的理解。

4. 马文·明斯基的"心智社会"

4.1 核心思想

“智能的力量源于我们巨大的多样性，而非任何单一的完美原则。”

对单一整体智能模型的反思：智能不是金字塔式的层级结构，而是扁平化、充满互动与协作的"社会"。

4.2 关键概念

概念 说明
智能体（Agent） 极其简单的、专门化的心智过程，自身"无心"（如 LINE-FINDER）
机构（Agency） 一组协同工作的智能体，完成更复杂任务
涌现（Emergence） 复杂智能行为从简单局部交互中自发产生

搭积木塔示例：BUILD-TOWER → BUILDER → ADD-BLOCK → FIND-BLOCK/GET-BLOCK/PUT-ON-TOP → SEE-SHAPE/REACH/GRASP。没有任何智能体拥有全局规划，但协作涌现出智能行为。

4.3 对多智能体系统的启发

  • 去中心化控制：无中央控制器，协调机制成为核心课题
  • 涌现式计算：蚁群算法、粒子群优化等
  • 智能体的社会性：通信语言、交互协议、协商策略、信任模型

5. 学习范式的演进

5.1 联结主义（Connectionism）

自下而上的方法，灵感源于大脑神经网络：

对比维度 符号主义 联结主义
知识表示 显式符号规则 分布式连接权重
获取方式 手工编码 从数据中学习
核心机制 逻辑推理 反向传播调整权重
优势 可解释 模式识别、鲁棒
劣势 脆弱、知识瓶颈 黑箱、不可解释

5.2 强化学习（RL）

专注于序贯决策问题的学习范式，通过"试错"学习最大化长期收益。

要素 说明
智能体 学习者和决策者
环境 智能体交互的对象
状态（S） 环境在某一时刻的描述
行动（A） 智能体可采取的操作
奖励（R） 环境反馈的标量评价信号
策略（π） 从状态到行动的映射

学习目标：最大化累积奖励（回报），而非即时奖励。

AlphaGo：通过数百万次自我对弈，从胜负结果中学习最优落子策略，不依赖人类棋谱。

5.3 预训练-微调范式

阶段 说明
预训练 在海量通用文本上通过自监督学习（预测下一个词）训练模型
微调 用少量特定任务标注数据适配下游任务

涌现能力（模型规模跨越阈值后出现）：

  • 上下文学习：Few-shot/Zero-shot 完成新任务，无需调整权重
  • 思维链推理：输出逐步推理过程，显著提升复杂推理准确性

LLM 不再仅仅是语言模型，而是兼具海量知识库和通用推理引擎双重角色的组件。

5.4 LLM 驱动的现代智能体

核心组件架构：

模块 功能
感知模块 从环境接收输入，形成观察
规划模块 反思、自我批判，分解宏观目标
LLM 深度推理，决策工具调用
记忆模块 整合历史信息
执行模块 解析指令，调用工具
工具箱 代码执行器、搜索引擎、API 等

6. 三大思潮与融合

思潮 代表 核心思想 里程碑
符号主义 西蒙、明斯基 符号操作与逻辑推理 SHRDLU、专家系统、深蓝
联结主义 辛顿 神经网络模拟大脑 反向传播、CNN、Transformer
行为主义 — 与环境互动试错学习 TD-Gammon、AlphaGo

21 世纪 20 年代：三大思潮深度融合。LLM（联结主义产物）成为执行符号推理、工具调用和规划决策的"大脑"，形成神经-符号结合的现代智能体架构。

7. 本章要点总结

  1. 符号主义以 PSSH 为理论基石，专家系统和 SHRDLU 是其典型成果
  2. 符号主义面临知识获取瓶颈、常识问题、框架问题、系统脆弱性四大挑战
  3. ELIZA 揭示了规则驱动系统的核心矛盾：看似智能，实则无理解
  4. "心智社会"理论提出智能从简单智能体的协作中涌现，启发了多智能体系统
  5. 联结主义用分布式权重替代显式规则，赋予智能体从数据中学习的能力
  6. 强化学习通过"试错"让智能体学会序贯决策
  7. 预训练-微调范式解决了知识获取瓶颈，LLM 展现出涌现能力
  8. 现代 LLM 智能体是符号主义、联结主义、行为主义三大思潮的深度融合
